### 主函数**TEST.cpp**（类外实现）
```cpp
#include<iostream>
#include"student.h"
using namespace std;
int main()
{
    student s; 
    s.name="Quxin";
    s.age=20;
    s.scroe=99;
    s.Display();
    cin.get();
}          
```
### 头文件**student.h**
```cpp
#pragma once
class student
{
public:
    const char* name;
    int age;
    int scroe;
    void Display();//没有函数体，类的声明
};
```
### 类文件**student.cpp**
```cpp
#include "student.h"
#include <iostream>
using namespace std;
void student::Display()
{
    cout<<name<<' '<<age<<endl;
    cout<<scroe<<endl;
}     
```
---
需要配置`task.json`
```js
"args": [// g++ 编译器后的参数
                "-fdiagnostics-color=always",//彩色警告
                "-g",//产生调试信息
                ///////////源文件位置////////
                "${file}",
                "student.cpp",
                ////////////////////////////
                "-I.",//添加头文件路径--当前目录
                "-o",//编译时的输出路径
                "${fileDirname}\\bin\\${fileBasenameNoExtension}.exe"//编译输出的可执行文件
            ],
```
